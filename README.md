## نصب

```sh
git clone https://github.com/mansourahmadi/tabchi
cd persianteam
chmod +x bot
./bot install
# and send [y] to finish install
```
* * *
## ساخت ربات
```
./bot create
./bot 1
وارد کردن ایدی عددی سودو#
وارد کردن شماره ربات#
```
## ساخت ربات های بیش تر

```sh
cd persianteam
./bot create
./bot number of bot >> شماره رباتی که به شما میدهد
وارد کردن ایدی عددی سودو#
وارد کردن شماره ربات#
```
